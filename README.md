# ☕ Brew Haven Coffee Shop

Brew Haven is a modern coffee shop website designed to provide customers with a seamless and enjoyable digital experience. The project focuses on creating an elegant and user-friendly interface that allows visitors to explore menus, reserve tables, discover promotions, and learn more about the coffee shop.

## 🎯 Project Goals

- Create a modern and premium coffee shop website.
- Improve customer engagement through intuitive navigation.
- Showcase menu items and promotions effectively.
- Simplify the table reservation process.
- Deliver a responsive and visually appealing user experience.

## ✨ Features

### Home Page
- Attractive hero section
- Featured menu showcase
- Promotional banners
- Customer testimonials

### Menu
- Browse coffee, tea, desserts, and snacks
- Product cards with images and pricing
- Search and category filtering

### Reservations
- Select date and time
- Choose number of guests
- Reserve tables online

### About Us
- Coffee shop story
- Brand values and mission
- Team introduction

### Reviews
- Customer ratings
- Testimonials and feedback

### Contact & Location
- Business information
- Opening hours
- Interactive location section
